# Como Iniciar o Projeto

Este documento serve como um guia rápido para você lembrar como abrir e rodar o projeto do Pet Spa.

## 1. Como abrir o projeto no VS Code

### Opção A: Pelo Terminal / Prompt de Comando
1. Abra o PowerShell ou Prompt de Comando do Windows.
2. Digite o seguinte comando e aperte Enter:
   ```bash
   code C:\Users\prisc\projects\pet-bath-scheduler
   ```

### Opção B: Pelo menu do VS Code
1. Abra o VS Code.
2. Clique em **File** (Arquivo) > **Open Folder...** (Abrir Pasta...).
3. Encontre a pasta `C:\Users\prisc\projects\pet-bath-scheduler` e selecione-a.

---

## 2. Como rodar o site localmente

1. Dentro do VS Code, abra o terminal integrado (`Terminal > New Terminal`).
2. Digite o comando abaixo e aperte Enter:
   ```bash
   npm run dev
   ```
3. Abra o seu navegador (Chrome, Edge, etc.) e acesse o endereço:
   [http://localhost:3000](http://localhost:3000)

Sempre que você quiser ver o site funcionando, é só repetir o Passo 2!
